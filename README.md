# Marc Castellví Vila — Portfolio 🎬

Portfolio of Marc Castellví, Compositing Artist & Lighting Artist.
Bilingual (EN/ES) home page with interactive project reel, plus
standalone pages for SEO: about, projects (one page per project),
reel, blog and contact.

Live at: https://marc-castellvi.com

## Structure

- `index.html` — home page (hand-maintained, inline CSS/JS).
- `about/`, `projects/`, `projects/<slug>/`, `reel/`, `blog/`,
  `blog/<slug>/`, `contact/` — static inner pages generated by
  `tools/build_pages.py` (edit the data there and re-run
  `python3 tools/build_pages.py`).
- `assets/` — shared stylesheet (`site.css`), project thumbnails and
  the Open Graph card (`og-card.png`).
- `media/` — local MP4 clips used by the interactive reel.
- `sitemap.xml`, `robots.txt` — crawling/indexing. Add new URLs to the
  sitemap when adding pages.

## SEO checklist when publishing new content

1. Add the page URL to `sitemap.xml` (with `lastmod`).
2. Give it a unique `<title>` + meta description + canonical URL.
3. Add descriptive `alt` text to every image.
4. Submit the sitemap in Google Search Console after deploying.
